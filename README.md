# network-traffic-monitoring

## Objective
To monitor and analyze network traffic using packet capture tools.

## Tools Used
- Wireshark
- Bettercap
- Kali Linux

## Steps Performed
1. Start packet capture
2. Filter traffic (HTTP, DNS, etc.)
3. Analyze packets

## Key Filters
http
dns

## Output / Results
- Captured HTTP requests
- Observed DNS queries
- Identified traffic patterns

## Conclusion
Traffic monitoring helps understand network behavior and detect suspicious activity.
